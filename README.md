# crm
客户管理系统
一套SpringBoot+Vue的客户管理系统。相关文档可进入doc目录下查看（后期会计划同步过来）
 ### 技术选型
 #### 后端：</br>
 1. 核心框架：SprinBoot 2.3.0.RELEASE
 2. 安全框架: SpringSecurity
 3. 持久层框架: MyBatis
 4. 日志管理：SLF4J
 #### 前端：
 1. Vue全家桶
 2. Element-UI
 3. 脚手架：Vue-cli3
 
 #### 开发环境
 1. JDK1.8
 2. MySQL 5.7
 3. Intellij IDEA(需要装Lombok插件)+WebStorm
 4. Maven
